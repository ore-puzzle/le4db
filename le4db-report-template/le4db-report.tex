\section{機能・インタフェース}

各機能および対応するインタフェースの説明を行い，典型的な状態のブラウザ画面キャプチャ図を付ける．
それぞれの画面でどのテーブルが関係しているか，発行されるSQL文の例などを記載する．
各機能ごとに小節(subsection)を作って記載するとよい．

なお，LaTeXにおいてブラウザ画面のキャプチャ図を挿入するには，例えば以下のような方法がある．

\begin{enumerate}

\item 画像の保存

[PrintScreen]キー(アクティブなウィンドウのみをキャプチャするには[Alt]+[PrintScreen]キー)\footnote{キーボードによっては[PrtSc]となっている．}を押し，画像ビューア/エディタ(GIMPやIrfanView等を用いることができる)に貼り付け([Ctrl]+[V])，PNG形式で保存する\footnote{EPS形式で保存可能な場合はEPS形式を用いるとよい．EPS形式の場合，バウンディングボックスファイルの作成は不要となる．}．ここで保存された画像を sample.png とする．

\item バウンディングボックスファイルの作成

sample.png を保存したディレクトリにて以下のコマンドを実行し，sample.bbファイルを作成する．

\begin{verbatim}
ebb sample.png
\end{verbatim}

\item 画像の挿入

.pngファイルおよび.bbファイルを同一のディレクトリに配置し，以下のような記述を用いて図を挿入する．挿入された図のサンプルは図\ref{fig:sample}のようになる．

\begin{verbatim}
\begin{figure}[tp]
\begin{center}
\includegraphics[scale=0.8]{sample.png}
\end{center}
\caption{サンプル．}
\label{fig:sample}
\end{figure}
\end{verbatim}

\begin{figure}[tp]
\begin{center}
\includegraphics[scale=0.8]{sample.png}
\end{center}
\caption{サンプル．}
\label{fig:sample}
\end{figure}

PNG画像を取り込んだPDFファイルの作成にはdvipdfmxコマンドを用いるとよい．

\end{enumerate}
